# 📺 IPTV System - Plataforma Profissional de IPTV

Sistema completo de gerenciamento de IPTV com autenticação, dashboard e API RESTful.

## 🚀 Funcionalidades

- ✅ Sistema de cadastro e login de usuários
- ✅ Dashboard personalizado com URL da playlist
- ✅ API compatível com formato antigo (`get.php`)
- ✅ Processamento automático de playlists M3U
- ✅ Atualização automática via GitHub Actions
- ✅ EPG integrado com múltiplas fontes
- ✅ Suporte a múltiplos players (VLC, Kodi, IPTV Smarters)
- ✅ Cache e otimização de performance
- ✅ Logs de acesso dos usuários

## 📋 Pré-requisitos

- Python 3.10+
- GitHub Account
- Render Account (ou similar)
